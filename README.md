# ⚡ Eletrocenter - Vitrine Digital

O **Eletrocenter** é uma landing page moderna para uma loja física de eletroeletrônicos. O projeto foi desenvolvido para funcionar como uma vitrine online, apresentando produtos, serviços e informações de contato de forma clara e responsiva.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi construído com as tecnologias fundamentais do Front-End:

- **HTML5**: Estruturação semântica (uso de header, nav, section, article e footer).
- **CSS3**: Estilização avançada com **CSS Grid** e **Flexbox** para layouts adaptáveis.
- **JavaScript (Vanilla)**: Manipulação de DOM para interatividade do menu mobile e botão de topo.

## 🛠️ Funcionalidades de Destaque

- **Layout Responsivo**: O site se adapta automaticamente para celulares, tablets e desktops.
- **Menu Mobile**: Menu lateral (hambúrguer) funcional para facilitar a navegação em telas pequenas.
- **Grid de Produtos**: Exibição organizada de produtos usando `grid-template-columns`.
- **Botão Voltar ao Topo**: Experiência de usuário (UX) aprimorada com navegação suave.
- **Mapa Interativo**: Localização da loja física via Google Maps.
